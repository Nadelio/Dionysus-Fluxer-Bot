Dionysus is the god of a bunch of stuff in Greek mythology, but the main thing I remember about them (him?) is that they party and drink a lot (you can really tell I've only ever played Hades).\
So I made named a minigame bot after him.\
It's a pretty simple bot, just a few games rn, will probably add more later.\
Making it open-source in case you want to self-host it for your own server. Yk following the good ol' FOSS standard that Fluxer itself follows :3

Anyways, please enjoy! Here is some information about the bot interface:

Command prefix: `!`

Commands:
- `help`, prints the list of avaliable commands
- `games`, prints the list of avaliable games
- `balance`, view the command author's balance
- `wins`, view the command author's wins in a specific game
- `daily`, claim your daily reward (or see how long until the next time you can claim it)

Games:
- Ping Pong (`ping`), mainly just used to test if Dionysus is awake and moving, but it also has a counter of how many times you've bothered the Dionysus \:)
- Craps/Dice (`dice`), the first stage of Craps (look it up), you can place a bet with the chance to double it if you win
- Rock, Paper, Scissors (`rps`), should I even explain this?
- Coin Flip (`coin_flip`), flip a coin and bet on the face-up side, if you win, get double your bet
- Word Scramble (`scramble`), guess the 5-letter word from its scrambled up form
- Guess The Number (`guess`), guess the secret number from 1 to 100
